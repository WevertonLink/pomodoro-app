# Tech Stack Documentation

## Frontend Framework
React 18 + Vite + TypeScript
- Fast HMR
- Optimized builds
- Type safety with strict mode

## UI Components
shadcn/ui + Tailwind CSS
- Copy-paste components
- Fully customizable
- Accessibility built-in

Key components: Button, Dialog, Dropdown, Tabs, Switch, Select, Progress, Toast, Card

## State Management
Jotai - Atomic state with localStorage persistence

## Routing
React Router v6 - Routes: /, /stats, /settings, /tasks

## Animations
Framer Motion - Smooth transitions and gestures

## Audio Management
Web Audio API + Custom Library
- Preload sounds on init
- Volume control
- Fade effects

## Date/Time
date-fns - Lightweight, tree-shakeable

## PWA
vite-plugin-pwa with Workbox
